# Тестовое задание — Продуктовый аналитик Фарпост

## Задачи

### Задача 1 — Дизайн A/B теста (раздел Автозапчасти)
Оценка эффекта от смены вида отображения объявлений: **список → плитка**

- Дизайн теста: гипотеза, метрики, разбивка трафика, длительность
- Расчёт размера выборки и статистической мощности
- Способы оценки результата

📁 `notebooks/01_ab_test_design.ipynb`

---

### Задача 2 — Дашборд метрик (раздел Недвижимость)
Проектирование дашборда для регулярного мониторинга здоровья продукта

- Выбор метрик с обоснованием
- Визуализация и структура дашборда
- Фильтры и период агрегации
- Гипотезы для анализа

📁 `notebooks/02_dashboard_design.ipynb`  
📁 `dashboard/` — макеты и референсы
